    # 2. print 相关

    # 1.大整数科学输出

    print （"%e" %number）可以将number输出为科学计数法
    这里的 number 是要转换的数字或者变量
    把科学记数法转化为十进制整数：print （"%d" %number）

    print ("%e"%1010)
    1.010000e+03

    print("%d"%1.010e+03)
    1010

    x=1010
    print("%x"%x)
    3f2

    print("%d"%1.2345e+03)
    1234

    print("%f"%1.2345e+03)
    1234.500000

    # 2.print格式

    1.输出指定位数小数

    方法一：round(X, N)

    该方法并不严格有效，当X小数位数n<N时，仅能够输出n位小数。

    方法二：print('%.Nf'%X)或者print("%.Nf"%X)

    注意该方法有两个“%”，没有“，”。

    方法三：print(format(X, '.Nf')或者print(format(X,".Nf")

    注意该方法没有"%"，但有“，”。

    # 7. 注意 -> copy的对象问题

    1.copy只会复制父对象，而子对象是公用的
    deepcopy是完全复制，所有的“父子”对象都会被完全复制

    例如：

    import copy
a = [1,2,[3,4]]
b = a
id(a) == id(b)    # Ture
b = copy.copy(a)
id(a) == id(b)    # False
a[2][0] = 123
a                 # [1,2,[123,4]]
b                 # [1,2,[123,4]]
b = copy.deepcopy(a)
a[2][0] = 123
b                 # [1,2,[3,4]]
